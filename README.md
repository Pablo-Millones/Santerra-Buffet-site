# Santerra Buffet - Propuesta de Rediseño

> Propuesta de rediseño desarrollada con fines demostrativos. No corresponde al sitio oficial.

## Funcionalidades

- **Chef Virtual de IA** - Genera sugerencias personalizadas de pastas
- **Sistema de Reservas** - Formulario con confirmación por WhatsApp y email
- **Galería Multimedia** - Fotos y videos con lightbox interactivo
- **Menú Responsivo** - Hamburger menu animado para móvil
- **Mapa Interactivo** - Ubicación en Google Maps
- **Smooth Scroll** - Navegación suave entre secciones

## Tech Stack

- **Frontend:** HTML5, Tailwind CSS
- **Backend:** Python
- **IA:** NVIDIA NIM API (Llama 3.1)
- **WhatsApp:** API directa con mensajes pre-armados

## Ejecutar

```bash
python server.py
```

Abrir http://localhost:8080

## Aviso

Este proyecto es una propuesta de rediseño con fines demostrativos y de portafolio. No está afiliado ni es representativo del sitio oficial de Santerra Buffet.
